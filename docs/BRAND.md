# Brand Assets

## Concept

EvoMap Console uses a folded route mark: a map path, three live nodes, and a terminal prompt line. The intent is to make the app feel like a native operator console for graph, skill, service, and node workflows without copying EvoMap's official brand mark.

## Asset Files

- Primary app icon PNG: `assets/brand/evomap-console-logo.png`
- Primary vector logo: `assets/brand/evomap-console-logo.svg`
- Wordmark vector: `assets/brand/evomap-console-wordmark.svg`
- macOS app icon set: `App/Assets.xcassets/AppIcon.appiconset/`

## Palette

- Deep console ink: `#0F1F2E`
- Map teal: `#7DE0D5`
- Agent blue: `#2CA6FF`
- Credit amber: `#F4B85C`
- Warm open-source paper: `#F6F3EA`

## Usage Rules

- Use the square icon for the macOS app, GitHub avatar, favicon, and compact docs surfaces.
- Use the wordmark for README headers, release notes, and social previews.
- Keep the icon text-free. Add text outside the mark when needed.
- Do not present this as an official EvoMap logo unless EvoMap explicitly approves that positioning.
- For open-source distribution, describe the project as an independent/operator console unless the relationship changes.
